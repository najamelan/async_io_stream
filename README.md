## Install
With [cargo add](https://github.com/killercup/cargo-edit):
`cargo add ws_stream_io`

With [cargo yaml](https://gitlab.com/storedbox/cargo-yaml):
```yaml
dependencies:

   ws_stream_io: ^0.1
```

With Cargo.toml
```toml
[dependencies]

    ws_stream_io = "0.1"
```
