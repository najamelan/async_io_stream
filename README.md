## Install
With [cargo add](https://github.com/killercup/cargo-edit):
`cargo add async_io_stream`

With [cargo yaml](https://gitlab.com/storedbox/cargo-yaml):
```yaml
dependencies:

   async_io_stream: ^0.1
```

With Cargo.toml
```toml
[dependencies]

    async_io_stream = "0.1"
```
